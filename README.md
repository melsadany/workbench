# Data Science Utilities

A collection of R, Bash, and other scripts for various data science tasks including file conversion, polygenic risk scoring, visualization, and statistical analysis.

## Categories

### File Conversion
- Format conversion between nifti to data frame in R.
- Genomics file format handlers
- Data preprocessing utilities

### Genetics
- PGS calculation pipelines
- Score normalization methods
- BED conversion to RDS

### Visualization
- Custom ggplot2 themes
- Plot formatting utilities
- Publication-ready figure templates

### Statistical Functions
- Specialized statistical tests
- Model evaluation metrics
- Implementing RUV-based approach
- Archetype analysis summary
- Comparing statistical models
- 
- Data transformation tools

### MRI Functions
- Extracting TR from nifti
- Extract slice-timing information
- Overlay mask and background image

### misc
- Get word embedding
- Run facial landmarking and get facial coords
